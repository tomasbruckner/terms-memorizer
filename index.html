Term memorizer

To use this website, you need to write terms in format: TERM - DESCRIPTION. One term per line. It is very important to insert - and only have one term and description per line (press enter for next line). Then press Update terms. After you update terms, you can press Random term to start memorizing the terms. If you cannot remember the term, click on Show term description to see the description. If you think you already memorize, you can delete current term with Delete current term button.

Example:

свой - moje, naše, vaše
ну - že jo, no
под - pod, pro, na
где - kde
дело - práce, činnost, dílo
есть - jíst
сам - sám, osobně
